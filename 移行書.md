- [ ] `rendering/src/rhi_renderer.cpp` を新規作成

**`initialize()`** (`sceneGraphInitialized` 相当・Qt が管理するタイミングで呼ばれる):

```
1. m_rhi = rhi()  // QQuickRhiItemRenderer::rhi() で QRhi* を取得（所有しない）
2. m_renderTarget = renderTarget()  // QQuickRhiItem が管理するオフスクリーン RT
3. QRhiBuffer (VertexBuffer / UniformBuffer) を生成・create()
4. QRhiShaderResourceBindings を生成・create()
5. QRhiGraphicsPipeline を生成・setRenderPassDescriptor() / create()
   ※ PSOCache::init() を呼び出す（Phase 6-1 完了後に有効化）
6. QRhiResourceUpdateBatch で頂点データをアップロード
```

**`synchronize()`** (GUI thread → render thread のデータ同期 / render thread で呼ばれる):

```
1. ECS の最新フレーム番号・パラメータを受け取る
2. GUI thread 側の状態を render thread 側のメンバにコピーする
```

**`render()`** (毎フレーム render thread で呼ばれる):

```
1. QRhiResourceUpdateBatch でユニフォームバッファを更新
2. Compute Pass（エフェクト適用）:
   cb->beginComputePass(resourceUpdates)
   cb->setComputePipeline(...)
   cb->setShaderResources(...)
   cb->dispatch(x, y, 1)
   cb->endComputePass()
3. Graphics Pass（Quad 描画 → オフスクリーン RT）:
   cb->beginPass(renderTarget(), clearColor, depthStencil, nullptr)
   cb->setGraphicsPipeline(m_pipeline.get())
   cb->setShaderResources()
   cb->setViewport(...)
   cb->setVertexInput(...)
   cb->draw(6)  // Quad = 2 triangles
   cb->endPass()
4. update() を呼んで次フレームをスケジュール（アニメーション再生中のみ）
```

### 付録A: Qt 6.7 未満での代替実装（`beforeRendering` + `beforeRenderPassRecording` 方式）

Qt 6.7 未満を対象とする場合は `QQuickRhiItem` が利用できないため、以下の方式を用いる。

```cpp
// PreviewCanvas : public QQuickItem
void PreviewCanvas::windowChanged(QQuickWindow *w) {
    if (!w) return;
    // リソース更新（バッファアップロード等）は beforeRendering で行う
    connect(w, &QQuickWindow::beforeRendering,
            m_renderer, &RhiRenderer::frameStart, Qt::DirectConnection);
    // 描画コマンドの記録はレンダーパス内の beforeRenderPassRecording で行う
    connect(w, &QQuickWindow::beforeRenderPassRecording,
            m_renderer, &RhiRenderer::mainPassRecordingStart, Qt::DirectConnection);
    // クリーンアップ
    connect(w, &QQuickWindow::sceneGraphInvalidated,
            m_renderer, &RhiRenderer::cleanup, Qt::DirectConnection);
}
```

`RhiRenderer::frameStart()` 内で `window()->rhi()` の nullptrチェックと初回初期化を行う。`QRhiCommandBuffer` は `QSGRendererInterface::CommandListResource` 経由で取得する。

各シェーダーの Vulkan-style GLSL Compute Shader 形式:

```glsl
#version 430

layout(local_size_x = 8, local_size_y = 8) in;

layout(binding = 0) uniform sampler2D inputTex;
layout(binding = 1, rgba8) uniform writeonly image2D outputTex;
layout(binding = 2) uniform Params {
    /* エフェクト固有パラメータ */
} params;

void main() {
    ivec2 id    = ivec2(gl_GlobalInvocationID.xy);
    ivec2 size  = imageSize(outputTex);
    if (id.x >= size.x || id.y >= size.y) return;

    vec2  uv    = (vec2(id) + 0.5) / vec2(size);
    vec4  color = texture(inputTex, uv);
    /* エフェクト固有の処理 */
    imageStore(outputTex, id, color);
}
```
